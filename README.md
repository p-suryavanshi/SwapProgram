# Swap Program

This program takes two integers from user input, swaps them using a temporary variable, and displays the swapped values. It demonstrates a simple concept of variable manipulation in Java. 

## Functionality
1. Input: The user is prompted to enter two integers.
2. Swapping: The program utilizes a temporary variable to hold the value of one integer while it assigns the other value to it.
3. Output: After the swapping is completed, it displays the new values of the integers.
